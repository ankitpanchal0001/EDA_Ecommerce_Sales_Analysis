# E-Commerce Sales Analysis | Exploratory Data Analysis (EDA)

An end-to-end Exploratory Data Analysis (EDA) project performed on an E-Commerce sales dataset to uncover customer purchasing patterns, sales trends, product performance, and business insights using Python.

The objective of this project is to transform raw sales data into actionable business insights through data cleaning, visualization, and statistical analysis.

---

#  Project Overview

Data-driven decision making is essential for every e-commerce business. This project explores an e-commerce sales dataset to identify hidden trends, customer behavior, and business opportunities that can improve sales performance.

The project follows the complete data analysis workflow from data preprocessing to business recommendations.

---

#  Objectives

- Understand the structure of the dataset
- Clean and preprocess raw data
- Handle missing and duplicate values
- Perform Exploratory Data Analysis (EDA)
- Identify sales trends and customer behavior
- Visualize important business metrics
- Generate actionable business insights
